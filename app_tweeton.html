features

Based on the official Twitter App, Tweeton had the main features of any twitter client.
Targeting iOS 4, Tweeton was a pre-ARC application.
Tweeton was release in April 2011. Was retired in 2014.

Technical features

  • Core Data
  • Twitter API
  • MGTwitterEngine
  • OAuthConsumer
  • GSTwitPicEngine
  • TouchJSON
  • MBProgressHUD
  • Offline support (tweets, images)
  • Async network calls
  • Async image loads
