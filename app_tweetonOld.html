Tweeton

Deprecated

Based on the official Twitter App, Tweeton had the main features of any twitter client.

Targeting iOS 4, Tweeton was a pre-ARC application.

Tweeton is not available anymore due to the changes in the Terms and Conditions from Twitter regarding 3rd party integrations.

Technical features

  • Persistence: Core Data
  • Twitter API
  • MGTwitterEngine
  • OAuthConsumer
  • GSTwitPicEngine
  • TouchJSON
  • MBProgressHUD
  • Local content to increase performance (tweets, images) and allow offline access
  • Async network calls
  • Async image loads
